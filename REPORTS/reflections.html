What was the most challenging part of this project for you? Were you able to overcome it?

I found setting up the portal page the most difficult part. Between making the folders containing the repositories git submodules and accidently adding .zip files to the repository and staging it. The portal page definetly gave me the most trouble. I was able to solve it mostly through copilot and I ended up learning a few more things because I was typing out the commands copilot gave me instead of copying and pasting.

What did you enjoy most about this project?

Honestly, I found making the frosted flakes box really intersting. Because I had to look up how to transform the text to change it's skew and orientation and I liked testing which angles looked closest to the box.

What would you do differently if you could do the project again?

I would pick more reliable teamates. In all the projects I'm part of except one I'm the only one who made any significant contributions to the project.

Do you feel more confident in working on real-world projects after this experience?
Explain.

I do feel more confident in working on real-world projects because I learned more about how to look things up and research things I don't/didn't know how to like using text-shadow to make an outline of the text or transforming the text to be rotated to an angle. I think these projects has given me the experience to make real-world html and css websites.
